Exploratory Data Analysis

Working with Datetime

  • Datetime conversion:
    • pd.read_csv('data.csv', parse_dates = ['date'])
    • df['date'].astype('datetime64')
    • pd.to_datetime(df['date'])
      • Secara default, pandas melakukan parsing pada tanggal yang ambigu (02-01-2020. 11-10-2020, etc.) dengan format bulan terlebih dahulu (YMD/ MDY):
        • Jika kolom memiliki format tanggal terlebih dahulu (YDM/DMY):
          • pd.to_datetime(df['date'],dayfirst=True)

  • Datetime extraction:
    • Jika kolom sudah bertipe data datetime, kita dapat mengakses dt accessor untuk mengambil informasi waktu dari kolom:
      • dt.month, dt.week, dt.day...
      • dt.to_period('M'), dt.to_period('Q')...

Working with Category

  • Category memiliki nilai "Categories" yang tersimpan

  • Setiap analisis yang kita lakukan akan mengacu pada nilai "Categories" dari kolom tersebut

  • Jika kolom sudah bertipe data category, kita dapat mengakses cat accessor untuk mengakses/mengubah "Categories" dari kolom tersebut:

    • data['column'].cat.reorder_categories(new_order): mengubah urutan dari kategori (contoh: ["Senin","Selasa","Rabu",...])

Tables

Frequency Table

  • data['col'].value_counts():

    • umumnya digunakan untuk eksplorasi cepat
    • nilai yg dikembalikan berupa pd.Series
    • Parameter tambahan yang dapat digunakan:
      • normalize=True get percentage/proportion instead of actual frequency.
      • sort=True sort by frequency
      • ascending=True sort in ascending order

  • pd.crosstab(index, columns)

    • Parameter tambahan:
      • normalize: mengubah frekuensi menjadi persentase
        • 'all'/True: proporsi berdasarkan keseluruhan data
        • 'index': proporsi berdasarkan baris
        • 'columns': proporsi berdasarkan kolom
      • margins:True: menambahkan kolom & index total (All)
      • margins_name: merubah nama 'All' dari margins
# simplest approach: using value_counts()

household['sub_category'].value_counts()

# more versatile: use `pd.crosstab(index, column)`
pd.crosstab(
    index = household['sub_category'],
    columns = "jumlah"
)

pd.crosstab(
    index = household['sub_category'],
    columns = household['format']
)

Aggregation Table

  • pd.crosstab(index, columns, values, aggfunc):
    • values: kolom numerik yang ingin dihitung
    • aggfunc: fungsi agregasi (sum, mean, median, max, dst.)
pd.crosstab(
    index = household['sub_category'],
    columns =  household['format'],
    values= household['quantity'],
    aggfunc = "sum"
)
  • pd.pivot_table(data, index, columns, values, aggfunc):
    • Memiliki parameter yang sama dengan crosstab
    • Perbedaannya:
      • paramater data: data tidak perlu di definisikan berulang saat mingin mengisi index/columns/dsb.
      • parameter index & columns: tidak lagi bersifat wajib (jika hanya ingin menghitung satu kategori, cukup dipilih salah satu)
      • parameter values: tidak lagi bersifat wajib, jika tidak diinput, akan memunculkan aggfunc adari setiap kolom numerik dari data.
      • parameter aggfunc: tidak lagi bersifat wajib, secara default akan dihitung rata2 (mean)
# contoh 1
household.pivot_table(
    index = "sub_category",
    values = "unit_price"
)

# contoh 2
pd.pivot_table(
    data = data_tangkap,
    index = "alat_tangkap",
    columns = "kategori_tangkapan",
    values = "jumlah_tangkapan"
)

Missing Values & Duplicates

Duplicated Data

  • Gunakan .duplicated()untuk mendeteksi baris yang duplikat:

    • df.duplicated()
      • keep=: 'first'/'last'/False
    • Kombinasikan dgn subsetting method:
      • household2[household2.duplicated(keep=False)]

  • Gunakan .drop_duplicates() untuk membuang nilai duplikat:

    • keep=: 'first'/'last': one of the occurence, False: drop all duplicates.

Missing Values

Detect NA:

  • Gunakan .notna() untuk mendeteksi nilai yang tidak kosong
  • Gunakan .isna() untuk mendeteksi nilai yang kosong:
    • df.isna()
    • df.isna().sum() --> menampilkan total baris yang kosong di setiap kolom
    • Kombinasikan dgn subsetting method:
      • df[df['col'].isna()] --> for NA in specific column.
      • df[df.isna().any(axis = 1)] --> filter NA in any column.

NA deletion:

  • Gunakan .dropna() untuk membuang baris dengan nilai NA:
    • df.dropna()
      • how=: any/all
      • thresh=: n of columns with NA values
  • Kombinasikan .notna()dgn subsetting method:
    • df[df['col'].notna()]

NA imputation:

  • Gunakan .fillna()untuk mengisi nilai yang kosong:
    • df['col1'].fillna('Missing')
    • df['col2'].fillna(0)
    • df['col3'].fillna(df['col3'].mean())
    • ..
      atau
    • df['col'].fillna(method)
      • method=:
        • bfill
        • ffill
