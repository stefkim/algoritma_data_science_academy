Day 1

Virtual Environment Preparation

  • Environment adalah lingkungan ketika kita bekerja menggunakan python. Environment sendiri terdiri dari python interpreter dan beberapa package/library.

  • Kernel adalah penghubung/konektor antara environment dan third-party (Jupyter notebook, Jupyter Lab, dll)

  • Package adalah pustaka/library yang dibutuhkan untuk melakukan task tertentu selama kita bekerja dengan python

  • Cek apakah Anaconda sudah terinstal dengan perintah

    conda --version

  • Jika perintah conda tidak dikenali, tambahkan direktori Anaconda ke PATH atau gunakan Anaconda Prompt

  • Membuat environment baru menggunakan perintah

    conda create --name <ENV_NAME> python=<PYTHON_VER>

atau 

conda create -n <ENV_NAME> python=<PYTHON_VER>

  • Melihat environment yang telah dibuat menggunakan perintah

    conda env list

    untuk melihat semua daftar environmentnya. Tanda asterik(*), menandakan environment yang sedang aktif di gunakan

  • Untuk mengubah ke aktif environment dapat menggunakan

    conda activate <ENV_NAME>

  • Untuk install kernel pada environment yang baru, kita butuh untuk menginsal ipykernel package. Gunakan perintah

    conda install ipykernel

    pada environment yang baru dibuat

  • Untuk mengkonfirmasi dan mendaftarkan kernel di environment baru gunakan perintah:

    python -m ipykernel install --user --name <KERNEL_NAME>

    untuk enable kernel pada environment tersebut

  • Untuk menonaktifkan virtual environment bisa menggunakan perintah :

    conda deactivate

Jupyter Notebook Shortcuts

Anda dapat menggunakan ctrl + shift + p untuk melihat semua shortcut yang dapat digunakan.

Ada 2 mode di jupyter notebook: command mode dan edit mode. Pada command mode Anda dapat melakukan:

  • b untuk membuat cell baru
  • dd untuk menghapus cell
  • c untuk mengcopy cell dan v untuk paste cell
  • y mengubah cell ke code cell
  • m mengubah cell ke markdown cell
  • enter untuk mengaktifkan edit mode

Pada edit mode Anda dapat melakukan:

  • ctrl + enter untuk run cell
  • ctrl + / untuk menambahkan command pada code cell
  • esc untuk mengaktifkan command mode

PANDAS

Instalasi : pip install pandas atau conda install pandas
Cara import library -> import <PACKAGE_NAME>. Contoh import pandas

  • as/alias : nama/alias yang digunakan untuk mendeskripsikan nama lain dari library

Ada 2 Tipe pada pandas:

  • Dataframe : Kumpulan dari Series yang memiliki baris dan kolom
  • Series : 1 kolom pada dataframe (1 dimensi)

Untuk membaca data :

  • read_csv() -> diperhatikan untuk hierarki file yang dibaca

  • head() : inspeksi 5 data teratas
  • tail() : inspeksi 5 data terbawah
  • dtypes : atribut untuk melihat tipe data pada dataframe

Method vs Atribut : secara kasat mata bisa dibedakan dengan melihat apakah ada tanda () di bagian akhir. Contoh:

dataframe.dtypes # contoh atribut
dataframe.head() #contoh method
  • Membuat dataframe : pd.DataFrame({'column_name' : [values]})
  • Membuat Series : pd.Series([values])

Data Types, Categorical & Numerical

Tipe Data yang ada di Pandas

  • string : object
  • categories variabel : category
  • integer : int64
  • floating point(decimal) : float64

Ada 2 alasan mengubah tipe data category :

Mengubah tipe data -> astype()
Misal:

df['format'] = df['format'].astype('category)

Subsetting and Indexing

Indexing dan subsetting digunakan untuk menyeleksi/memfilter/memilih sebagian data dari keseluruhan data yang ada di dataframe. Tekhnik indexing dan subsetting yang bisa dilakukan dengan pandas adalah:

  • select_dtypes(), digunakan untuk menyeleksesi kolom(beberapa kolom) dengan tipe data tertentu
rice.select_dtypes('category') # memfilter semua kolom yang tipe datanya category
  • drop(), secara default akan menghapus baris yang telah di deskripsikan di dalam method. Method ini juga bisa digunakan untuk menghapus kolom dengan menambahkan parameter axis=1.
rice.drop(5) #menghapus baris dengan nama index 5
rice.drop(column='receipt_id', axis=1) # menghapus kolom receipt_id
  • [] operator, square bracket digunakan untuk melakukan subset data menggunakan index/row dari dataframe (index python dimulai dari 0). Operasinya dijelaskan dengan [start:end]
rice[1:5] # ini akan menampilkan baris ke 2 sampai dengan ke 5 pada dataframe rice
  • .loc, adalah atribut yang dapat digunakan untuk memilih baris dan kolom. Syntaxnya adalah .loc[baris,kolom] dimana jika Anda ingin mengambil semua elemen baris atau kolom, Anda dapat menggunakan tanda :. .loc sendiri merujuk pada nama row dan nama kolom.
rice.loc[:,['receipt_id', 'purchase_time']] # menampilkan semua baris serta kolom receipt_id dan kolom purchase_time.
  • .iloc sama seperti .loc adalah atribut yang dapat digunakan untuk memilih baris dan kolom. Syntaxnya adalah .iloc[baris,kolom] dimana jika Anda ingin mengambil semua elemen baris atau kolom, Anda dapat menggunakan tanda :. .iloc sendiri merujuk pada lokasi index row dan nama kolom.
rice.loc[:,0:2] # menampilkan semua baris serta kolom 1 sampai kolom 2

Conditional Subsetting

Conditional Subsetting umunnya digunakan untuk memfilter data berdasarkan kondisi tertentu yang dipakai untuk kebutuhan analisis. Misalnya:

  • Kita akan mengambil data yang transaksinya berasal dari supermarket -> .format=='supermarket'
  • Kita akan mengambil transaksi produk yang harganya lebih dari 10000 -> .unit_price>10000
  • Kita hanya akan mengambil semua transaksi yang pembelian barangnya tidak sama dengan 0 -> .quantity!=0, dll

Untuk melakukan filtering menggunakan conditional subsetting, Anda dapat menuliskan syntax sebagai berikut:

df[df[column]=='kondisi'] # 1 kondisi
df[(df[column]=='kondisi 1')operator(df[column]=='kondisi 2')] # 2 atau lebih kondisi

Penggunaan tanda kurung () diperlukan untuk memisahkan antar kondisi jika Anda memnggunakan lebih dari 1 kondisi.

Referencing and Copying

Referencing digunakan untuk membuat sebuah dataframe baru yang merefer ke suatu dataframe tertentu. Jika kita mengubah nilai pada dataframe baru maka nilai pada dataframe lama juga akan ikut berubah.

Catatan untuk refer: 2 dataframe dengan nama yang berbeda tetapi hanya merefer 1 objek dataframe yang sama.

Copying digunakan untuk menduplikat objek dataframe. Jika kita mengubah nilai pada dataframe baru maka nilai pada dataframe lama tidak akan ikut berubah.

Catatan untuk copy: 2 dataframe dengan nama yang berbeda sekaligus dikenali sebaga 2 objek dataframe yang berbeda pula.
